# UCLLaTeXPosterTemplate
A LaTeX template for the new A0 UCL brand academic poster. To use the correct fonts, you will need to download and unzip the UCL Sans font pack in the project folder - this can be found at <https://www.ucl.ac.uk/brand-and-experience/brand/brand-resources>.

Other than this, the template should be fairly self-explanatory - if there are any issues with this template, feel free to put an issue on the Github page.
